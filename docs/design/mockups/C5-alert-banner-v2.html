C5 — Alert Banner v2 — National System Colors

Redesigned alert banner component for the Clear Skies Now page. Each banner adapts its color treatment to the source national alert system — not a generic palette. JMA, MeteoAlarm, UK Met Office, NWS, IMD, BoM each have their own official color schemes. OWM passthrough alerts receive neutral treatment (no severity data available). Shows collapsed state, expanded state, multi-alert stack, light theme, and dark theme.

Design Notes & Decisions

Color signal: The severity-colored icon block is the primary color anchor. The card border matches it at 2px. The glass text area stays neutral (never tinted with the severity color) so text contrast is always predictable.

National system specificity: JMA uses purple (level 4) and deep purple/black (level 5) — distinct from MeteoAlarm red. UK Met amber is a warmer orange than MeteoAlarm orange. These are NOT interchangeable.

OWM passthrough: Advisory yellow (#C8A400), generic ph:warning icon, "Severity Unknown" badge label, role="status". Treated as level-2 per operator direction — an alert that exists is important enough for a visible color signal. Note: the research doc (§7) recommended neutral treatment to avoid misrepresenting unknown severity; this design decision departs from that recommendation.

ARIA semantics: Level 4–3 alerts use role="alert" (live region, announced immediately). Level 2 and lower use role="status" (polite). OWM passthrough always role="status". This matches the urgency signal.

Expand button: The entire collapsed row is a button (tabindex="0", role="button", aria-expanded). The expanded panel has role="region" and an id referenced by aria-controls. Keyboard accessible — Enter/Space to toggle.

Icon sizing: 2.25rem (36px) desktop, 1.875rem (30px) mobile. Much larger than the 20px mentioned in previous banners — must be immediately recognizable, especially for emergency-level events.

Yellow text on icon: Yellow-background icon blocks (advisory/yellow/watch levels) use dark text (#3d2800) on the icon for WCAG AA contrast, not white.

Badge contrast — dark theme: Dark-theme badge backgrounds are higher opacity (0.30 vs 0.14 light) and badge foreground uses lighter hue variants so the text-on-dark-glass badge meets AA contrast.
